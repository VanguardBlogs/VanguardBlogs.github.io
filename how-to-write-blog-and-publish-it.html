How to write a blog and publish it?




Conditions: N/A

Article Average Read Time : Approximately 5 minutes.

Greetings!

Welcome to the step-by-step guide on how to write a blog and publish it!

By the end of this article, you would know how to write a blog, the key features to be kept in mind while writing a blog, and how can you bring your content to the web!



What is a blog?

What is a blog?

According to the Oxford English Language Dictionary, “A blog is a regularly updated website or web page, typically one run by an individual or small group, that is written in an informal or conversational style.” and it's true. A blog is a regularly updated website or web page, which means that your content should be put online to be called a blog.

The steps involved in blogging are as follows:

1/3: Choosing the theme



The first and the most important step, choosing the theme.



The first step in writing a blog is most crucial in the process.

I have struggled to choose the correct theme before and after starting this website, but after thinking for a few days, I tried to write an article and found that it was pretty good, so I choose technology as my blogging genre.

What's yours?
From the vast menu of blogging genres, it is difficult to choose one. Try your best to choose the one which suits you!

2/3: Choosing the name of your blog



Choosing the name is another important step in the process.

A catchy name would catch the sight of the reader and attract them to your website. Make sure that the name you choose, would be catchy, as well as an ideal name that suits your theme and content.



3/3: Choose your website domain/blog service



If you are good at programming, creating a blog page would not be much big of a task for you, there are two more things that you need to keep in mind while making your website, domain name, and the color of the content of your website. While choosing the domain name, you will need to make sure that you choose the name with matches the name of your blog, because, if there would be no connections between the domain name and blog title, the reader would be confused. With engaging content, the colors you choose matter too!



Now, the second half of this point, choosing your domain/blog service provider.

If you choose the domain option, you will need to do the following things:

> Buy a domain
> Configure it on a domain hosting site
> Learn to code in mainly HTML and CSS (You can skip CSS to use an existing template that is available on the internet)
> Upload files using an FTP server

Sounds tough?

Try the second method!

The second method, as said above, is, using a blogging platform.

It is very easy to use, one such platform is Blogger by Google.

All you need to do is, enter your content and then add images of your choice.

Going back to the domain option,

It has certain benefits and some drawbacks for beginners.

Benefits:

> Completely customizable.
> Infinite color choices.
> Infinite font-styles.
> Add many other features using JavaScript such as "Go back to top" and "Dark Mode".

> Drawbacks for beginners:

> Add your website to Google Search Console
> Programming might be difficult for beginners
> Looking after the Search Engine Optimization Score (SEO Score)
and others

Now, it's up to you, which path would you choose?
The easy-peasy blogging service? or the domain option?

Take your time.
Don't make any decision in haste.
Don't take the stress for writing content every day, go on at your own pace. : )



Thank you for reading!
